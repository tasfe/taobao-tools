	V_1.0功能说明
		1.提供给卖家管理客户，可以挖掘潜在客户
		2.对客户进行分类，比如：
			发掘客户
				10天购买一次商品以上为潜在客户
				15天购买2次商品以上为
				30天购买3次商品以上为新客户
				... ...
			流失型客户
				30天未购买商品为预流失客户
				45天为购买商品为流失客户
		3.短信功能
			可以拉拢潜在客户，找回流失型客户，可以设定定时任务，定时发送优惠信息的短信给潜在客户。
			也可以通过其它途径挖掘潜在客户
		4.提供报表功能
			统计客户每月新增数量
			统计每月营销数额
			... ...

盈利模式
	1.按月收费
	2.按年收费
	3.按店铺等级收费
	4.可以赚取发送短信的差额费用
	... ...
	可以免费试用一个月
